Gunpo Eunhye Corps Family Management System v6.2

[공지사항 업데이트]
1. 공지사항.xlsx 파일을 엽니다.
2. '공지사항' 시트의 5행부터 내용을 입력합니다.
3. 열 이름과 순서는 변경하지 않습니다.
4. GitHub 저장소에서 기존 공지사항.xlsx 파일을 새 파일로 교체합니다.
5. Commit changes 후 앱의 공지 메뉴에서 새로고침을 누릅니다.

공지사항 열
시작일 / 종료일 / 구분 / 제목 / 내용 / 중요
- 중요 Y: 중요공지로 강조
- 중요 N: 일반공지
- 시작일~종료일에 해당하는 공지만 표시

[교회일정 업데이트]
1. 교회일정.xlsx 파일을 엽니다.
2. '교회일정' 시트의 5행부터 내용을 입력합니다.
3. GitHub 저장소에서 기존 교회일정.xlsx 파일을 새 파일로 교체합니다.
4. Commit changes 후 앱의 일정 메뉴에서 새로고침을 누릅니다.

교회일정 열
날짜 / 시간 / 구분 / 행사명 / 장소 / 담당 / 비고
- 오늘 이후 일정만 날짜순으로 표시

[출석 통계 개선]
- 오늘 이후의 미래 날짜 출석 CSV는 통계와 그래프에서 자동 제외합니다.
- 배포용 ZIP에는 테스트 출석 CSV가 포함되지 않습니다.

[GitHub 업데이트]
