# GunpoEunhye Corps Family Management System V8.4.0 Final Stable

## 운영 원칙
- GitHub Pages에서 무료로 운영합니다.
- 배포 ZIP에는 테스트 출석 CSV가 포함되지 않습니다.
- 실제 출석 데이터는 GitHub 저장소에 직접 올린 파일만 사용합니다.

## 실제 출석 파일 형식
예:
- attendance-2026-07-12-성결예배.csv
- attendance-2026-07-14-새벽기도.csv
- attendance-2026-07-15-수요예배.csv
- attendance-2026-07-17-금요기도회.csv

## GitHub 업데이트 방법
1. 이 ZIP을 압축 해제합니다.
2. 기존 GitHub 저장소에서 `Add file` → `Upload files`를 누릅니다.
3. 압축을 푼 프로그램 파일을 업로드합니다.
4. `Commit changes`를 누릅니다.
5. 기존 실제 출석 CSV는 삭제하지 않습니다.

## 공지와 일정
- 공지사항.xlsx
- 교회일정.xlsx

현재 안정화 버전에서는 공지와 일정 내용을 수정한 뒤 파일을 다시 업로드해 앱 업데이트 ZIP을 생성하는 방식이 가장 안전합니다.
